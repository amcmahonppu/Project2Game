Github Guardians Api

This API is for the Github Guardians project. The project builds a world for the player to explore, battle, and collect items in. When you include rpg.js a global variable is created called partyorganizer. All methods should be called on this object. For example, we can set the type of the party by calling the rpg.chrCreate2(name, hp, mp, att, def, matt, mdef) Below you can find the API:
